- For any suggestions regarding this project or the website, feel free to send them to [i@himeno-sena.com](mailto://i@himeno-sena.com). Friendly messages of any kind are also welcome — replies are casual and whenever the mood strikes.
- 对于本项目、网站有任何好的建议都可以发送到邮箱 [i@himeno-sena.com](mailto://i@himeno-sena.com)，同时邮箱欢迎任何友好的消息，随缘回复
- このプロジェクトやサイトへのご意見・ご提案は [i@himeno-sena.com](mailto://i@himeno-sena.com) までお気軽にどうぞ。どんな気軽なメッセージも歓迎します。返信はその気になったときに、ゆるりと。
