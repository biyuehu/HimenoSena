## 简体中文

收录范围包括但不限于：

- 诋毁、辱骂、攻击、讽刺、抹黑**姬野星奏**
- 诋毁、辱骂、攻击、讽刺、抹黑、造谣**新岛夕**
- 诋毁、辱骂、攻击、讽刺、抹黑、差评**恋×シンアイ彼女**及新岛夕其它主要作品

提交方式与 [留言](./messages.md) 类似，此处不具体赘述。

可按照以下格式发送邮件：

- 收件人：[i@himeno-sena.com](mailto://i@himeno-sena.com)：
- 主题：姬野星奏网站黑名单
- 内容：

```txt
平台：填写平台名字，如 QQ、Bilibili、小黑盒、Steam、Bangumi 等
Id：用户的唯一标识符，取决于具体平台，如 Bilibili 则是 UID，QQ 则是 QQ 号
原因：请尽量详细
```

此外，也可以 Fork 仓库编辑 `/data/blacklist.raw.json` 文件（不是 `/data/blacklist.json` 文件），在末尾追加新数据并发起 PR。

## English

The blacklist covers, but is not limited to:

- Defaming, insulting, attacking, mocking, or slandering **Himeno Sena**
- Defaming, insulting, attacking, mocking, slandering, or spreading false information about **Niijima Yuu**
- Defaming, insulting, attacking, mocking, leaving malicious criticism of, or slandering **Koi x Shinai Kanojo** and other major works by Niijima Yuu

The process is similar to submitting a [message](./messages.md) — no need to repeat the details here.

You may send an email in the following format:

- Recipient: [i@himeno-sena.com](mailto://i@himeno-sena.com)
- Subject: Himeno Sena Website Blacklist
- Body:

```txt
Platform: Name of the platform
Id: The user's unique identifier on that platform
Reason: Please be specific
```

Alternatively, you may fork the repository and edit the `/data/blacklist.raw.json` file (not `/data/blacklist.json`), append the new entry at the end, and open a PR.

## 日本語

収録範囲は以下を含みますが、これに限りません：

- **姫野星奏**への誹謗・侮辱・攻撃・皮肉・中傷
- **新島夕**への誹謗・侮辱・攻撃・皮肉・中傷・デマの流布
- **恋×シンアイ彼女**および新島夕の他の主要作品への誹謗・侮辱・攻撃・皮肉・中傷・悪意ある批評

手順は [メッセージの投稿](./messages.md) と同様ですので、ここでは詳しく繰り返しません。

以下のフォーマットに従ってメールをお送りください：

- 宛先：[i@himeno-sena.com](mailto://i@himeno-sena.com)
- 件名：姫野星奏サイト ブラックリスト
- 本文：

```txt
プラットフォーム：プラットフォーム名を記入
Id：そのプラットフォームにおけるユーザーの一意識別子
理由：できるだけ詳しく記入してください
```

また、リポジトリをフォークして `/data/blacklist.raw.json` ファイル（`/data/blacklist.json` ではありません）を編集し、末尾に新しいデータを追記して PR を作成する方法もあります。
